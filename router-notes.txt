- npx create-react-app router 


-----------------------------------------------------------------------------------------------------


Step 2: React projemize react-router-dom'u indiriyoruz.
cd router

- npm i react-router-dom


-----------------------------------------------------------------------------------------------------


Step 3: Folder Structure

- To create an application with multiple page routes, let's first start with the file structure. Within the 
src folder, we'll create a folder named pages with several files:

src\pages\:

Layout.js
Home.js
Blogs.js
Contact.js
NoPage.js

- Each file will contain a very basic React component.


-----------------------------------------------------------------------------------------------------


Step 4: index.js dosyasında react-router-dom'u ve componentleri (sayfaları) import ettik, kullandık.
